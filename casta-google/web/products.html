Web Site

We provide a very convenient way to create a web site

All web sites are saved and run on our own server. We provide a wide range of pre cooked components to help you build your web site easily.

It is possible to generate a full web site either from scratch or from templates using our user friendly web site editor. All web sites created are fully customizable.

You can contact us, we can help you to start building and customizing your web site. We can also create your web site completely from scratch for a very reasonable fee.

Features of web sites
  • Fully ajax (works very fast)
  • Very lightweight
  • Search engine friendly
  • Accordions, carousel,slide shows, gallery
  • Wide range of ready to use buttons
  • Google Map

  • Wide range of templates for articles

  • Icon and images gallery
  • Google Analytics

Web Site Editor

The most advanced website editor
The website editor is a WYSWYG tool to build or customize your web site. You can drag and drop components, save them and they are available online. It is completely web based. i.e. You can access it simply by using your favorit web browser. The tool comes with a wide range of ready to use components ranging from small text elements, articles template, forms, sliders, tables to more complete applications like organization management system, document management system. All these components are available right there at your finger tips. Just drag and drop and save.


DMS (Document Management System)

Very userfriendly and advanced tool to manage and share documents only
The document management system helps to manage documents throughout your organization efficiently. It comes with an intuitive user interface mimicking classical windows file explorer. It is also a content management system. Meaning that you can upload files, create folders, share folders among collaborators and even create and manage articles to be availabe on your website.

It comes with built in security system, full text search, WEB DAV protocol and Workflow management system

Catalogue Creator

Complete tool to create and configure products and services
Create and manage a beautiful online catalogue for your website. Our system comes with a complete inventory system for you to easily create your products and publish them on your online catalogue. Illustrate your products with photos, videos etc Add options to your products Specify product prices Manage and update stocks

Shopping Cart

Very convenient shopping cart
The component is made available via the online website editor. You just drag and drop it on your website, and it is ready to use. It is very beautifully designed, fully ajax. Built in currency calculator, connected to checkout system. It is used in conjuction with the catalogue. Once the shopping cart is activated, the "Add to cart" button becomes available in your catalogue. Users can add to cart and then check out.

Check Out System

Complete tool to create and configure products and services
We provide a secure checkout system built-in with the shopping cart. The checkout system is connected to a payment gateway and delivery system

Customer Relationship Management

Convenient tool to track and manage your customers
Our customer relationship management system is a useful application to manage clients that have registered through your website. With this application you can contact your clients by email, create and schedule newsletters and view your clients' information (e.g. clients' contact or payment information, products bought by clients etc...)

Sales Tracking System

Easily configure workflow to suit your chain of command
This application enables the management of orders received from clients via your website. It comes with a built in workflow for sales management. Orders can be viewed, approved, cancelled etc... Sales can be sorted by different colours according to their state (e.g on delivery, preparing, delivered etc) and can easily be managed

Organization Management System

The most advanced website editor
This application is used to create and manage the different users using the system. It comes with a built-in security system. You can restrict access to different applications or files of the web OS to specific groups of users. You can view logs of activity on the system for all users and identify unauthorised activity.
