Save the portal

Select start -> save portal

Note: only the portal will be saved

Note: in the developer version, you will not be able to create new portals.

Create new page

  1. Select start ->New Page
  2. Enter the page name
  3. The page will be opened automatically ready to add components and save
Note :Each page can be considered as an applications for developers.

For web master will be considered as normal web pages. But the concept is essentially the same.

Save a page

  1. To save a page select Start->save page
  2. Enter the page name
  3. The page will be opened automatically ready to add components and save
Note: Only the page will be saved not the whole portal

Open a page

  1. Select start -> open page
  2. From the mini file selector, select the page you want to open
