Default template

  • Create a template accessible via a url
  • Configure this component to point the template towards the url of the template
  • select a tag in your template you want to "dynamise", and give it a name (name=thename)
  • Drag a component from the left bar and drop it on the template
  • Change the name of the component to the name specified in the template
  • Now you can configure the added component the way you feel like
